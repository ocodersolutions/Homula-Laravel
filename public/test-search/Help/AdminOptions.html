Getting Started ››

After installing the software make sure that you've changed the permission of site.inc.php (linux command: chmod 755 site.inc.php ) back to normal i.e not writable by world.

Point your browser to the address where you've installed this software, for ex if you uploaded the entire folder to myWebsite.com then point your browser to http://www.myWebsite.com . Login as admin by using your admin username and password that you created during installation.
