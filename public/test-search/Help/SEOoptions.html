SEO options

Getting Started ›› Admin Options ››

This options are important for search engine optimization. You can specify the title, description and keywords.


1) Website title: This is the text that is shown in the title of your browser. This is also recommended as it also shown in search engine search results.


2) Homepage description: This is the description which is shown by the search engines when your website shows up in any search result.


3) Homepage keywords: This is helpful for search engines to find out the type of website. Usually search engines crawls site pages to find out more about the site. But this setting is recommended. So try to add keywords which are are related to your website.
