How to find what version I currently have?


You have:


Version 1.0: If you downloaded 'Real Estate Made Easy' script before 12-Feb-2012

Version 1.1: If you downloaded 'Real Estate Made Easy' script after 12-Feb-2012 and before 30-May-2012

Version 1.2: If you downloaded 'Real Estate Made Easy' script after 31-May-2012

Version 1.3: If you downloaded 'Real Estate Made Easy' script after 29-Sept-2012

Version 1.4: If you downloaded 'Real Estate Made Easy' script after 21-April-2013


You can also check the version by opening header.php in notepad etc and view the license information at the top.
