Top menu

Getting Started ›› Admin Options ››

The default option for each of the following field is empty which gives your website top menu the default look. But you may change any or all of them at your discretion. The colors of your website top menu would change in real time so you can judge which color combination looks the best.


1) Choose top menu background color:  Using this option you can set the background color of the top menu. Default is set to black.


2) Choose top menu border color: This would set the border color of the top menu.


3) Choose top menu font color: This would set the font color of top menu.
