Listing Options

Getting Started ›› Admin Options ››

1) Notify admin when a member posts a new listing or edit an existing one?: Enable this option if the site admin should get an email when someone posts or edits a listings.


2) Show email on the listing page?: Enable this option if contact email of members should be displayed on their listing page.


3) Listing minimum heading length (characters): Set the minimum character requirement for the headline on add/edit listing page. If you set it to 0, then it will become a non require field.


4) Listing minimum description length (characters): Set the minimum character requirement for the description on add/edit listing page. If you set it to 0, then it will become a non require field.
