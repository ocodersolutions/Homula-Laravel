System requirements

Getting Started ››

Following are the minimum requirements required for installing this software


PHP version: 5.x (allow_url_fopen or cURL module should be enabled in order for Google geocoding to work. SSL module such as openssl should be enabled and port 587 open, if you plan to use Gmail/Apps for emailing)


MySQL version 3.2x or newer


Web hosting


It should get installed without a problem in most of the cases. If you are not sure of the minimum requirements, then try installing and if you get error then ask your hosting service provider about how it can be fixed. You may also contact me through the comments page of code canyon.
