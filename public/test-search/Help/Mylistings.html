Getting Started ››

This page shows the listing that were posted by the member who is logged in currently. Other options such as editing/deleting would be shown using which a member can update the listings.

The site admin would see an extra option using which she can set the listing as a top or featured listing.

Top listings would be shown above any other listings on the site. Featured listings would be shown just below the top listings and have a different color row and also marked as featured.
